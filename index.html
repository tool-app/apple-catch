Fruit Catcher Game

Auf dem Bildschirm werden verschiedene Früchte in zufälliger Größe und Farbe erscheinen. Der Spieler muss auf die Früchte klicken, bevor sie den unteren Rand des Bildschirms erreichen. Jede erfolgreiche Klick auf eine Frucht wird dem Punktestand des Spielers hinzugefügt. Wenn eine Frucht den unteren Rand des Bildschirms erreicht, ohne dass der Spieler sie geklickt hat, ist das Spiel vorbei. Der Spieler kann das Spiel neu starten, indem er auf den "Neu starten"-Button klickt.
